# PulseForge-Studio
PulseForge Studio is a customizable MIDI editing workspace focused on speed, clarity, and playable workflows.

Core features:

**Piano roll editing with track, channel, and velocity control

Customizable movable workspace with detachable panels

Playable arrangement simplification and VP sheet conversion

Sheet music PDF export and source lookup tools

MuseScore preview bridge with optional imported userscripts

Fast editing flow for remixing, practice prep, and arrangement cleanup

Built from CPP and will not be open sourced due to specific features.**
